HTML5:Figure and Figure Caption

<figure></figure>

Images often come with caption HTML5 has introduced a new element to contain images and their caption so that the two are associated.
You can have more than image inside the element as long as they all share the same caption.

<figcaption></figcaption>

The element has been added to HTML5 is order to allow wed page authors to add a caption to an image.
Before these elements were created there was no way to associated an element with its caption.
Older browsers that do not understand HTML5 elements simply ignore the new elements and display the content of them.
